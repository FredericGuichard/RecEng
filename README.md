# RecEng
coupling between recycling and ecosystem engineering through detritus

# Description

- The main receng.py script reads all parameter files, initiates an instance of the recengClass class, loops over the parameter space, and store results on file
- The recengFunctions.py defines the recengClass class that initiates all variables and the main function receng()
- The auto_eng_stoch folder contains parameter files and ploting script to generate simulation results for the case of 4 ecosystem compartments where the primary producer is an ecosystem engineer and where both deterministic and stochastic simulations are produced

# Running the code

- Set the correct paths in receng.py to the parameter folder and execute receng.py.
- receng.py and recengFunction.py must be in the same directory
